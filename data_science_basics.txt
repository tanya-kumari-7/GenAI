
Data Science Basics
====================

What is Data Science?
---------------------
Data Science is the field of extracting meaningful insights from data using:
- Math & statistics
- Programming
- Domain knowledge
- Visualization
- Machine Learning

It answers questions like:
- Why are users dropping off?
- Which customers are likely to churn?

Core Components
===============

1. Data Collection
------------------
Collect data from:
- Databases (SQL, NoSQL)
- APIs
- Web scraping
- Logs, sensors

Tools:
- Excel, SQL
- Python (requests, pandas, BeautifulSoup)

2. Data Cleaning (Preprocessing)
--------------------------------
Fix or remove bad data:
- Missing values
- Duplicates
- Inconsistent formats

Example:
```python
import pandas as pd
df = pd.read_csv("data.csv")
df.dropna()
df.duplicated().sum()
```

3. Exploratory Data Analysis (EDA)
----------------------------------
Understand the data using:
- Summary statistics
- Visualizations
- Correlation

Tools:
- pandas, matplotlib, seaborn, plotly

```python
import seaborn as sns
sns.pairplot(df)
```

4. Data Visualization
---------------------
Make visual plots to see trends:
- Line charts, bar charts, histograms

Tools:
- Excel, Tableau, Power BI
- Python: matplotlib, seaborn

```python
df['sales'].plot(kind='bar')
```

5. Statistical Analysis
-----------------------
Understand patterns:
- Mean, Median, Mode
- Standard Deviation
- Hypothesis Testing (t-test, chi-square)

6. Machine Learning Basics
--------------------------
Model building for predictions.

Types:
- Supervised Learning (Regression, Classification)
- Unsupervised Learning (Clustering, PCA)
- Reinforcement Learning

Tools:
- scikit-learn, tensorflow, keras

```python
from sklearn.linear_model import LinearRegression
model = LinearRegression()
model.fit(X_train, y_train)
```

7. Model Evaluation
-------------------
Check model performance:
- Accuracy, Precision, Recall
- RMSE, MAE

8. Deployment
-------------
Put your model into production.

Tools:
- Flask, FastAPI
- Streamlit
- AWS, GCP, Azure

Example Project: Titanic Dataset
--------------------------------
Steps:
1. Load with pandas
2. Clean missing values
3. EDA (e.g., gender vs survival)
4. Logistic Regression model
5. Evaluate accuracy
6. Deploy with Streamlit

Skills Required
---------------
- Programming: Python, SQL
- Stats & Math
- Visualization
- Business understanding
- Communication
